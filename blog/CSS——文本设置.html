CSS——文本设置

发表于 2017-12-28 | 分类于 web笔记

这是一个月以前学习CSS时所做的笔记，在博客做好以后就想将它发表出来，以供今后查阅。

1. 字体属性

  • 字体类型： font-family: "微软雅黑", "Microsoft Yahei";
    用”,”隔开多个值将会从前往后找，直到找到匹配字体，找不到则为默认字体。如：p { font-family: Helvetica, Sans-Serif; }
  • 字体粗细（字重）：font-weight: xx(100~900之间的一个值);
  • 字体大小：font-size: xx px;
    -

    2. 文字属性

    • 对齐方式
      【 text-align: left(左对齐)/right(右对齐)/center(居中)/justify(两段对齐)/inherit (从父元素继承 text-align 属性的值); 】
    • 行高
      【 line-height: 1.3(文字倍数)/ 10px(像素表示); 】
    • 文字装饰
      【 text-decoration: underline(下划线)/overline(上划线)/line-through(删除线)/blink(闪烁文本)/none (无);】
    • 文字缩进
      text-indent:50px

3. display属性

1）显示为块级元素（与母元素同宽，padding填充上下有效）
display: block;
2）显示为行内元素（可随别的元素浮动，padding填充上下无效）
display: inline;
3）显示为行内块元素（既具有块级元素属性又可浮动）
display: inline-block;

4. 文本颜色

  • 十六进制值: ＃FF0000 ；如：color: #ff0000;
  • 一个RGB值: RGB(255,0,0)；如：background: rgb(255,0,0);
  • 颜色的名称: red；如：background-color: red;

5. 文字转换与缩进

  • uppercase 所有大写
  • lowercase 所有小写
  • capitalize 首字母大写

总结

所有可用的CSS文本属性：

  • direction 【设置文本方向】
  • letter-spacing【设置字符间距】
  • text-shadow 【设置文本阴影】
  • text-transform 【控制元素中的字母】
  • unicode-bidi 【设置或返回文本是否被重写 】
  • vertical-align 【设置元素的垂直对齐】
  • white-space 【设置元素中空白的处理方式】
  • word-spacing 【设置字间距】
